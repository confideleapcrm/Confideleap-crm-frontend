### Step 1: Download the Template
1. Save the `investor_import_template.csv` file to your computer

### Step 2: Prepare Your Data
Replace the sample data with your actual investor information. Here's what each column represents:

#### üë§ **Personal Information**
- **First Name** *(Required)*: Investor's first name
- **Last Name** *(Required)*: Investor's last name  
- **Email** *(Required)*: Primary email address
- **Phone**: Phone number with country code (e.g., +1 555-123-4567)
- **Job Title**: Current position (e.g., Partner, Principal, Managing Director)
- **Seniority Level**: Level within firm (Partner, Principal, Associate, etc.)

#### üè¢ **Firm Information**
- **Firm Name** *(Required)*: Investment firm or organization name
- **Firm Type**: Type of firm (VC, PE, Angel Group, Corporate VC, Family Office)
- **Firm Website**: Company website URL
- **Location**: Geographic location (City, State/Country)

#### üîó **Social & Professional Links**
- **LinkedIn URL**: Full LinkedIn profile URL
- **Twitter URL**: Twitter profile URL (optional)
- **Personal Website**: Personal or professional website

#### üìù **Professional Background**
- **Bio**: Brief professional background and investment focus
- **Education**: Educational background (separate multiple with semicolons)
- **Experience**: Professional experience (separate multiple with semicolons)

#### üí∞ **Investment Preferences**
- **Investment Stages**: Comma-separated list (e.g., "Series A, Series B")
  - Valid options: Pre-Seed, Seed, Series A, Series B, Series C, Series D+, Growth, Late Stage
- **Sector Preferences**: Comma-separated list (e.g., "FinTech, SaaS, AI/ML")
  - Valid options: FinTech, HealthTech, EdTech, SaaS, E-commerce, AI/ML, Biotech, etc.
- **Geographic Preferences**: Comma-separated regions
  - Valid options: North America, Europe, Asia Pacific, Latin America, etc.
- **Min Check Size (USD)**: Minimum investment amount in USD
- **Max Check Size (USD)**: Maximum investment amount in USD

#### üìà **Portfolio & Investments**
- **Portfolio Companies**: Comma-separated list of portfolio companies
- **Notable Investments**: Comma-separated list with details (e.g., "Stripe Series C, Coinbase Series B")

#### üè∑Ô∏è **CRM Information**
- **Status**: Current relationship status
  - Valid options: hot, warm, cold, contacted, unresponsive
- **Tags**: Comma-separated tags for categorization
- **Notes**: Additional notes or comments about the investor

### Step 3: Data Formatting Guidelines

#### ‚úÖ **Best Practices**
- Use consistent formatting for similar data types
- Include country codes for international phone numbers
- Use full URLs for websites and social profiles
- Separate multiple values with commas
- Keep text fields under 500 characters for optimal performance

#### ‚ö†Ô∏è **Common Mistakes to Avoid**
- Don't use special characters in email addresses
- Avoid empty rows in the middle of your data
- Don't mix currencies (use USD for check sizes)
- Ensure investment stages match the valid options exactly
- Don't include commas within individual cell values (use semicolons instead)

#### üìã **Required Fields**
The following fields are mandatory and must be filled:
- First Name
- Last Name
- Email
- Firm Name

4. **Import**: Complete the import process
